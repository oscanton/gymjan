- Arquitectura
  - Departamentacion y aislamiento
    - Datos -> .js
    - Estilos -> .css
    - Logicas -> .js
    - Estructura -> .html
  - Estructura de carpetas y archivos de la PWA
    - index.html y views/: estructura y paginas principales.
    - js/app.js: bootstrap e inicializacion de vistas.
    - js/core/: logica compartida (configuracion, calculos, almacenamiento, UI).
    - js/pages/: logica especifica de cada vista.
    - js/data/: datos estaticos (alimentos, menus, rutinas).
    - css/app.css y css/components.css: estilos globales y componentes.
    - manifest.json y sw.js: configuracion PWA y Service Worker.

- User Interface
  - Almacenamiento permanente
  - Instalacion PWA en smartphone
  - Paginas: Index, Calculadora, Actividad, Menu, Lista, Control
    - Renderizado general, mojibakes e idioma.
    - Exposicion de calculos.
    - Bootstrapping.
    - Usabilidad.
    - Apariencia.
